# Physical AI & Humanoid Robotics

Welcome to the comprehensive open-source course book on Embodied AI and Humanoid Systems.

## Course Modules

This course is organized into 6 comprehensive modules:

- [**Introduction & Why Humanoids Matter**](/docs/intro-and-why-humanoids) - Understanding the fundamentals of physical AI and humanoid robotics
- [**ROS 2: The Robotic Nervous System**](/docs/module1-ros2-nervous-system) - Mastering the Robot Operating System for humanoid control
- [**Gazebo & Digital Twin Simulation**](/docs/module2-gazebo-digital-twin) - Creating and simulating humanoid robots in virtual environments
- [**NVIDIA Isaac: The AI Robot Brain**](/docs/module3-nvidia-isaac-brain) - Leveraging AI for perception and decision-making
- [**Vision-Language-Action (VLA) Models**](/docs/module4-vision-language-action) - Integrating multimodal AI in humanoid systems
- [**Capstone: Autonomous Simulated Humanoid**](/docs/capstone-autonomous-humanoid) - Building a complete autonomous humanoid system

## About This Course

This course provides a comprehensive, practical introduction to humanoid robotics with a focus on real-world applications. Each module includes:

- Technical concepts and theory
- Real code examples that run on actual robots
- Mermaid diagrams for visual understanding
- Real robot spotlights with 2025 specifications
- Interactive quizzes for self-assessment

Start with the [Introduction](/docs/intro-and-why-humanoids) to begin your journey into humanoid robotics.
