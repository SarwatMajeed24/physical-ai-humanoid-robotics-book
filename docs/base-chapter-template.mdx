# Chapter Template Guide

This guide explains how to create course chapters with all the required structure elements for the Physical AI & Humanoid Robotics course book.

## Required Structure Elements

Each chapter should follow this structure:

### 1. Frontmatter
```md
---
title: Your Chapter Title
sidebar_label: Your Chapter Title
sidebar_position: X
description: Brief description of the chapter content
keywords: [physical ai, robotics, humanoid, specific-topic]
---
```

### 2. Imports
Import the required components at the top of your MDX file:

```md
import { ChapterLayout, ChapterSection, RobotSpotlight, QuizSection } from '@site/src/components/ChapterLayout';
import { MermaidDiagram, Citation } from '@site/src/components/EducationalComponents';
import { PythonCodeBlock, ROS2CodeBlock } from '@site/src/components/CourseChapter';
```

### 3. Chapter Layout
Use the ChapterLayout component to wrap your content:

```md
<ChapterLayout title="Your Chapter Title" description="Brief description">
  <!-- Your chapter content goes here -->
</ChapterLayout>
```

### Code Examples
Use the appropriate code block components:

```jsx
<PythonCodeBlock title="Python Code Example">
# This is a 5-15 line runnable Python/ROS 2 code snippet
import rclpy
from rclpy.node import Node

class MinimalPublisher(Node):
    def __init__(self):
        super().__init__('minimal_publisher')
        self.publisher = self.create_publisher(String, 'topic', 10)
        self.timer = self.create_timer(0.5, self.timer_callback)

    def timer_callback(self):
        msg = String()
        msg.data = 'Hello World'
        self.publisher.publish(msg)
</PythonCodeBlock>
```

```jsx
<ROS2CodeBlock title="ROS 2 C++ Code Example">
// This is a 5-15 line runnable C++/ROS 2 code snippet
#include <rclcpp/rclcpp.hpp>
#include <std_msgs/msg/string.hpp>

using std::placeholders::_1;

class MinimalPublisher : public rclcpp::Node
{
public:
  MinimalPublisher() : Node("minimal_publisher") {
    publisher_ = this->create_publisher<std_msgs::msg::String>("topic", 10);
  }

private:
  rclcpp::Publisher<std_msgs::msg::String>::SharedPtr publisher_;
};
</ROS2CodeBlock>
```

### Diagrams
Include Mermaid diagrams to visualize concepts:

```jsx
<MermaidDiagram chart="graph LR; A[Robot] --> B[Environment]; B --> C[Perception]; C --> D[Decision]; D --> A;" altText="Diagram showing robot-environment interaction loop" title="Robot-Environment Interaction" />
```

### Robot Spotlight
Highlight real robots with the RobotSpotlight component:

```jsx
<RobotSpotlight robotName="Example Robot" image="/img/robot-image.png" description="Description of the robot and its significance." />
```

### Knowledge Check
Include quizzes to test understanding:

```jsx
<QuizSection question="What is the primary function of a humanoid robot's perception system?" options={["To move the robot's limbs", "To interpret sensory data from the environment", "To generate power for the robot", "To store robot configuration data"]} correctAnswer={1} />
```

### References
Include proper citations:

```jsx
<Citation id={1} authors={["A. Author", "B. Author"]} title="Title of Research Paper" journal="IEEE Transactions on Robotics" year="2023" />
```

### Summary
End with a summary of key points.
